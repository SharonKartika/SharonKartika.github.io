Production

Main activity of a firm. The process of transforming inputs into output.

Production function gives the maximum output for different combinations of inputs.

The standard inputs are Labor (LLL) and Captial (KKK) and the output is QQQ.

Q=f(L,K) Q=f(L,K) Q=f(L,K)

Usually, we can write,

Q=ALαKβ Q=AL^\alpha K^\beta Q=ALαKβ

A is called the technology factor. α\alphaα and β\betaβ are positive.

Returns to scale

Measures the responsiveness of output to proportional change in input in all inputs.

Cob-Douglas Production function:

Q=f(L,K)=ALαKβ Q=f(L,K)=AL^\alpha K^\beta Q=f(L,K)=ALαKβ

If we increase both inputs ttt times. The function is then,

Q′=f(tL,tK)=A(tL)α(tK)β=tα+βALαKβ=tα+β⋅Q Q'=f(tL,tK) = A(tL)^\alpha(tK)^\beta\\=t^{\alpha+\beta}AL^\alpha K^\beta\\=t^{\alpha+\beta}\cdot Q Q′=f(tL,tK)=A(tL)α(tK)β=tα+βALαKβ=tα+β⋅Q

Thus, increase in input by a multiple of ttt gives an tα+βt^{\alpha+\beta}tα+β multiple of production.

Now, if α+β=1\alpha+\beta=1α+β=1, then input multiple of ttt gives output multiple of ttt. This case is called constant returns to scale. Half the input and the output halves as well. CRS.

If α+β>1\alpha+\beta>1α+β>1, then it is called increasing returns to scale. Input multiple of ttt gives output multiple greater than ttt.

If α+β<1\alpha+\beta<1α+β<1, then it is decreasing returns to scale. Input multiple of ttt gives output multiple less than ttt. We say that the production function returns to scale. DRS.

Cost of production

Cost function. Two inputs L,KL,KL,K with input prices www (wage), and rrr (price for capital, rent). If QQQ is the output, the cost function will then be,

C=C(w,r,Q) C = C(w,r,Q) C=C(w,r,Q)

rrr is the interest paid on the rent, but not the whole, since it asset. Similarly for equipment, which can later be sold.

Marginal cost is the increase in cost to produce an additional unit.

∂C∂q=C′(Q) \frac{\partial C}{\partial q} = C'(Q) ∂q∂C​=C′(Q)

Average cost (AC)

AC=CQ AC = \frac{C}{Q} AC=QC​

Plotting L(Q)L(Q)L(Q) against Q.

If the plot is linear, then the marginal cost is constant. In this case, the average cost is equal to marginal cost.
